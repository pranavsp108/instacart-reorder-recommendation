# Project Summary

Built a scalable recommendation engine using PySpark and Databricks on the Instacart dataset.

Key components:
- Reorder prediction model (GBT, PR-AUC: 0.41)
- Basket recommendations using FP-Growth
- Customer segmentation into 5 behavioral groups

Impact:
- ~4x lift over random baseline
- ~58% recall@10 for reorder prediction
- Actionable cross-sell and personalization strategies

Tech:
PySpark, Databricks, Python, Spark ML
